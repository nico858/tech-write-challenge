URL

https://8j5baasof2.execute-api.us-west-2.amazonaws.com/production/users/new

HTTP Method

POST

Response Type

Parameters

  • Query: username User/properties/username

    username that will be use to login

  • Query: password User/properties/password

    password that will be use to login
